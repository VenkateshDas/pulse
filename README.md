# Pulse

A native macOS command center for performance and storage.

> **Note**: This repository previously housed a Python TUI called `mac-monitor`. That code has been archived to `legacy_tui/`. The project is now focused entirely on the Swift desktop app, **Pulse**.

## Repository Structure & LLM Wiki

This repository is organized as an **LLM Wiki** to help both humans and AI coding assistants easily navigate, query, and maintain the project.

- **[Index](index.md)**: The catalog of all project components, documentation, and graphs.
- **[Pulse App](pulse/README.md)**: The source code for the macOS application.
- **[Docs](docs/)**: Product specifications and design mockups.
- **[Logs](log.md)**: The chronological log of major updates.
- **[Graphify Reports](pulse/graphify-out/GRAPH_REPORT.md)**: A generated knowledge graph of the Pulse app.
